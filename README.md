# Battle City

Little fun terminal-style game for MacOS.

## Project Structure
```
PROJECT_ROOT
├── assets/
│   ├── docs/
│   ├── imgs/
│   └── REFERENCE.md
├── utils/
├── main.cpp
├── README.md
└── LICENSE
```

## Installation
```shell
$ git clone git@github.com:1chooo/battle-city.git
```

## How to Use
```shell
$ cd battle-city
$ ./build.sh
```

#### How to generate `build.sh`

1. Modify the path in `build.sh`
2. Give permission to `build.sh` by:
    ```s
    $ chmod +x build.sh
    ```
3. Run in terminal:
    ```s
    $ ./build.sh
    ```
